# The-Developers-Arena_task-5
# 📊 Customer Sales Analysis Using Advanced Pandas

## 📌 Project Overview

This project analyzes customer purchasing behavior using **Python** and **Pandas**. It demonstrates advanced data manipulation techniques such as data cleaning, grouping, aggregation, filtering, merging, pivot tables, and visualization to generate meaningful business insights. The project also includes customer lifetime value analysis, sales performance evaluation, customer retention analysis, and an automated PDF report.

---

## 🎯 Objectives

- Analyze customer purchasing patterns.
- Calculate Customer Lifetime Value (CLV).
- Identify top-performing customers.
- Evaluate monthly sales performance.
- Analyze regional sales distribution.
- Measure customer retention.
- Discover cross-selling opportunities.
- Generate business insights through visualizations.

---

### Sales Dataset (`sales_data.csv`)
- Date
